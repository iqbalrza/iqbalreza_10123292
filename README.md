# 🛍️ Website Toko Elektronik

Tugas Besar mata kuliah Web Development — sebuah website toko elektronik yang menampilkan berbagai produk seperti smartphone, TV OLED, proyektor, speaker, dan aksesoris lainnya.

## 🖥️ Demo

Buka file `index.html` langsung di browser untuk melihat tampilan website.

## 🛠️ Teknologi

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white)

## 📁 Struktur File

```
├── index.html         # Halaman utama website
├── home.webp          # Gambar hero/banner utama
├── banner.png         # Banner promosi
├── s22.png / s23.png / s24.png  # Produk Samsung Series
├── oled.png           # Produk TV OLED
├── projectors.png     # Produk Proyektor
├── speakers.png       # Produk Speaker
└── ...
```

## 🚀 Cara Menjalankan

1. Clone repo ini:
   ```bash
   git clone https://github.com/iqbalreza/iqbalreza_10123292.git
   ```
2. Buka file `index.html` di browser favorit kamu.
